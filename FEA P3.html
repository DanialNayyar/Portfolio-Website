Derivation of 2D Quadrilateral Element Formulations for Finite Element Analysis

Derived different matricies used to make the shape function plots



🔧 Overview

This project involved deriving the Finite Element formulations for a 2D quadrilateral (4-node) using isoparametric interpolation. The project involved, shape function manipulation, jacobian transformation, strain-displacement and element stiffness matricicies which could be programmed for FEA implementation


🎯 Objectives

  • Derive Shape Functions Ni(ξ,η)
  • Compute Jacobian Matrix and its determinant
  • Derivation of Strain Displacement Matrix B
  • Derivation of Stiffness Matrix, K
  • Plot Shape Function (MATLAB)

Shape Function plot, N1

Shape Function Plot, N4

🛠️ Tools & Techniques

  • MATLAB: Shape Functions
  • Numerical Integration
  • Mapping between natural (ξ, η) and global (x, y) coordinates

📊 Key Results

  • Derived full shape function for 4-node quadrilateral element: 

    Ni4Q = α1i + α2iφ + α3iη + α4iφη

    And numerically determined shape functions: 
    N1 =  0.295 - 0.516φ - 0.516η + 1.180φη
N2 =  0.500 - 0.279φ + 0.131η - 2.818φη
N3 = -1.017 + 2.780φ + 2.780η - 4.068φη
N4 =  0.220 - 0.760φ + 0.240η - 1.721φη
  • Constructed the full strain-displacement matrix B using symbolic partial derivatives of each shape function.
  • Computed the Jacobian matrix J for isoparametric mapping and its determinant |J| to allow coordinate transformation.

    • 🧠 Reflections

    This project provided a deeper understanding of Finite Element from first principles. It linked the matrix assembly, coordinate transformation and interpolation theory.
