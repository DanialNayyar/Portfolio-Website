This project explored how different mesh element types accuracy of stress and strain values near a fracture tip under a load. A 3D specimen was designed in SolidWorks and meshed in ABAQUS using the different elements. Stress concentration, displacement and strain behaviour were plotted.

  • Investigation of mesh type around crack tip

ğŸ“Š Key Results

  • Tetrahedral Elements: Provided best result with respect to the curved edges (e.g. holes), it resulted in more accurate stress distribution at the tip
  • Hexahedral Elements: Highest local stress but poor fitting around the holes
  • Wedge Elements: Provided a balance between mesh density and fidelity with a slight reduction in accuracy
  • Principle Stresses were higher at the tip for all mesh types

ğŸ§  Reflections

This project demonstrated the importance of element selection in Finite Element Analysis (FEA) especially in important zones like crack tips. Mesh topology affected the stress predictions even though initial boundary conditions were the same. Tetrahedral elements demonstrated that the disadvantage of high accuracy is increased computational cost, as a result it is important to consider the accuracy-computational cost trade-off when selectin mesh element type for a given study.
